# 🧠 Agentic PowerShell CLI Assistant

A local, on-demand CLI tool that converts **natural language** into safe, validated PowerShell commands using **LangGraph multi-node workflows** with **Human-in-the-Loop (HITL)** approval.

---

## 🚀 Features

- 🗣️ Natural language → PowerShell translation
- 🔁 Multi-node LangGraph decision pipeline
- 🛑 Human-in-the-Loop approval before execution
- 🔴🟡🟢 Risk classification engine (HIGH / MEDIUM / LOW)
- 🔍 Scope validation (only PowerShell/system tasks)
- 🛠️ Intelligent error recovery with retry support
- 🔒 Stateless & privacy-first design
